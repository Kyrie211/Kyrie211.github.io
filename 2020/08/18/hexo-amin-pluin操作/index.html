hexo admin plugin

本地安装

// cnpm 是 npm 的国内镜像（淘宝），访问下载速度快很多
cnpm install --save hexo-admin

启动本地服务器

hexo server -d

访问在线编辑视窗

// 本地hexo博客地址 + admin即可
http://localhost:4000/admin
